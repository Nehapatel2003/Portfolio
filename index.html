Amazon Clone

Developed a responsive clone of the Amazon homepage using HTML5 and CSS3, replicating the design and layout of the world’s largest e-commerce platform. The project includes a navigation bar with search functionality, category menus, promotional banners, and product showcase sections.

Technologies used :
  • HTML5 for structure
  • CSS3 for styling (Flexbox, Grid, box-shadow, hover effects)
  • (Optional) JavaScript for interactivity in dropdowns or sliders

Weather App

Developed a weather detection website that allows users to enter the name of a city and instantly view the current weather conditions including temperature, location, date, time, and weather status (e.g., mist, cloudy, sunny). The app dynamically fetches weather data using a Weather API and displays it in a clean, user-friendly interface.

Technologies used :
  • HTML5 – structure of the page
  • CSS3 – styling and layout
  • JavaScript (ES6) – API integration, DOM manipulation, and event handling
  • OpenWeatherMap API – source for live weather data

Dictionary App

Developed a Dictionary Web Application where users can type any word and instantly get its definition, phonetics, audio pronunciation, examples, and part of speech. The app uses a Dictionary API to fetch real-time data and display it in a clean, user-friendly interface.

Technologies used :

Image Slider
